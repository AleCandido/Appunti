# Appunti
Questo repo è fatto per raccogliere un po' di appunti a caso: non avrò mai abbastanza tempo, voglia e continuità per scrivere da solo gli appunti di un corso, per cui non ci sono dispense complete.

In compenso chi vuole può trovare qualche appunto scritto qua e là per radunare le idee, e qualche nota di comodo (tipo struttura).

Il materiale è tutto libero, fatene ciò che vi pare.

## Struttura della Materia
Per leggere un file .md (Markdown) non ci vuole molto: al più bisogna installare un plugin sul proprio browser, ed è facilissimo.

Per i più pigri metto un paio di link per i browser più diffusi, ma ci sarà sicuramente un modo anche per tutti gli altri.

- [Firefox plugin](https://addons.mozilla.org/it/firefox/addon/markdown-viewer/)

- [Chrome plugin](https://chrome.google.com/webstore/detail/markdown-preview/jmchmkecamhbiokiopfpnfgbidieafmd)

NOTA: il browser è un programma come tutti gli altri, può leggere anche i file che stanno sul vostro computer, non solo quelli sul web

Per i pigrissimi, dato che qualcuno ha già manifestato interesse per questa lista, mi sforzerà di compilarla anche in html (c'è un programma in automatico che lo fa, [Pandoc](http://pandoc.org/)), però io lo scriverò in Markdown, quindi può darsi che il file html non sia sempre aggiornato quanto il Markdown.
